About

A free web-based tool designed for Ragnarok X: Next Generation players to calculate base multipliers and optimize stat distribution for maximum DPS output.

Why was this created?

The community spreadsheet was solid for quick checks, but only covered 3 stats. I added the remaining in-game stats for more accurate calculations that better match actual damage output

What are the features?
  • Base Multiplier Calculator
  • Multi-Target
    • Specific MVP/Mini
    • Average boss level per size
  • Stat Optimization Engine with randomized recommendations based on your stats
  • Equipment Set Bonuses including Vesper, Blue, and White sets
  • Flash Simulation for Spear of Eternity and Reaper Scythe
  • Progressive Web App with exclusive features when added to your home screen:
    • Quick access with a single tap from your home screen
    • Automatic saving of your calculated stats
    • Complete offline functionality
Is it safe?

This tool runs locally in your browser for instant results and complete privacy. No data ever leaves your device—no registration, no servers, no tracking.

The code is open source and available on the GitHub repository for full transparency.

How to Use
  1. Dropdowns
    • Select your Attack Type, Weapon, and Weapon Attribute.
    • For MVP/MINI:
      • Choosing a specific MVP/Mini will automatically sync and lock its size, race, and attribute. (For example: Phreeoni = Large / Brute / Neutral. These fields stay locked, you just need to fill in the numbers.)
        • If you only have a Race enchant (or only Attribute), set the other field to 0.
          • Example: Brute 0 / Neutral 20
          • Example: Brute 30 / Neutral 0
      • Choosing Average Lv. lets you manually set or skip the target race/attribute.
      • If you don't have race and/or attribute enchants, just go with Average Lv. Boss and skip those fields.
  2. Equipment
    • Select equipment set you're using (Vesper / Blue / White).
  3. Stat Inputs
    • Enter your stats exactly as they appear in-game.
    • Extra: Final DMG Bonus %
      • F. P/M DMG BONUS
        • Check detailed stats for Final Damage Stack / Final Damage Bonus. Make sure no buffs are active. If you don't see it, just set it to 0.
    • Only numbers, don't type "%".
    • Decimals are fine. (Example: 888.88)
  4. Calculate
    • Once everything's filled in, hit Calculate.
  5. Results
    • You'll see the base multiplier and a breakdown of your stats.
  6. Simulate Flash
    • Spear of Eternity and Reaper Scythe.
      • Each has 10s duration / 20s cooldown with +84% or +28% bonus. This tool normalizes them to 42 and 14 for 100% uptime in flash bonus contribution. But it still uses the original values for calculations.
  7. Swap Target
    • Lets you switch bosses without starting over.
    • If the new boss has different size/race/element, the inputs unlock and you'll need to re-enter your stats.
    • If it's the same type, the calculation updates instantly.
    • Examples:
      • Phreeoni 140 → Deviling 150 → re-enter Medium / Demon / Shadow.
      • Phreeoni 140 → Phreeoni 150 → auto updates, no changes needed.
      • Phreeoni 150 → Avg. 140 → race/element are cleared, treated as average Large Boss Lv.140.
  8. Recommendations
    • At the bottom you'll see a table with recommended stat allocations.
    • Hit Refresh to generate new sets until you find one that works for you.
